# Installation

```
sudo apt install libglfw3 libglew-dev glew-utils libglew2.0 assimp-utils libassimp-dev libassimp3v5 freeglut3-dev libglm
mkdir build
cd build
cmake ..
make -j4
./parkviewer
```
